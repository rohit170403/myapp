# Android App with Python

A simple Android app built with Python using Kivy and KivyMD frameworks.

## Features
- User registration
- User login
- Dashboard after authentication
- Material Design UI

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
python main.py
```

## Build for Android

1. Install Buildozer:
```bash
pip install buildozer
```

2. Build APK:
```bash
buildozer android debug
```

## Usage
1. Register a new account
2. Login with your credentials
3. Access the dashboard after successful login
